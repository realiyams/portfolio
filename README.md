# My Portfolio

Website portfolio pribadi yang dibangun menggunakan Next.js untuk menampilkan proyek, keahlian, dan pengalaman saya sebagai Fullstack Developer.

Website ini berisi kumpulan proyek pilihan, teknologi yang digunakan, serta informasi kontak untuk keperluan kerja sama atau profesional.

---

## Fitur

- Tampilan modern menggunakan Next.js App Router
- Desain responsif (desktop & mobile)
- Navigasi dengan smooth scrolling
- Animasi interaktif
- Struktur kode yang rapi dan mudah dipelihara

---

## Teknologi yang Digunakan

- Next.js
- React
- TypeScript
- Tailwind CSS
